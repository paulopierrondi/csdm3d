# Launch Plan

## Positioning

CSDM3D is a community-built visualization and maturity accelerator for ServiceNow professionals.

Use language that complements ServiceNow:

- "built with ServiceNow APIs"
- "helps explain CSDM maturity"
- "supports workshops and maturity conversations"
- "does not replace ServiceNow CMDB, Discovery, Service Mapping, Now Assist, or governance"

Avoid language that competes with ServiceNow:

- "replacement"
- "better than ServiceNow"
- "autonomous remediation without governance"
- "official ServiceNow tool"

## LinkedIn execution

- Use the first line as a hook, not an introduction.
- Put the GitHub link after the value explanation, not in the first sentence.
- Use a 45-60 second video with captions.
- Start the video directly on the problem or 3D map, not with a logo animation.
- Ask a specific closing question to invite comments.
