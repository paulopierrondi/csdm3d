# Video Script

Target: 45-60 seconds.

Format: vertical or square for LinkedIn feed. Use burned-in captions because many viewers watch with sound off.

## Storyboard

1. Hook, 0-3s

Text on screen: `CSDM maturity should not live in spreadsheets.`

Voiceover: "CSDM maturity should be visible, explainable, and actionable."

2. Product reveal, 3-10s

Text on screen: `CSDM3D`

Voiceover: "So I built CSDM3D, a public ServiceNow community app."

3. Login and connect, 10-20s

Text on screen: `Login. Connect ServiceNow. Analyze CSDM5.`

Voiceover: "You log in, connect a ServiceNow instance, and run a lightweight CSDM5 analysis through APIs."

4. Map moment, 20-35s

Text on screen: `3D maturity map`

Voiceover: "The output is a 3D maturity map across Foundational Data, Design, Build, Technical Services, and Sell/Consume."

5. AI value, 35-48s

Text on screen: `AI-ready insights`

Voiceover: "The value of AI starts with an explainable CMDB. CSDM3D turns maturity signals into insights and next actions."

6. CTA, 48-60s

Text on screen: `Open source on GitHub`

Voiceover: "I am open-sourcing this for the ServiceNow community. Download it, test it, fork it, and tell me what you would add next."

## CTA options

- "Download it on GitHub."
- "Fork it and use it in your next CSDM conversation."
- "Tell me which CSDM domain you would improve first."
